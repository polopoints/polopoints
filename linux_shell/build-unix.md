UNIX BUILD NOTES
====================
Some notes on how to build polopoints  in Unix. 

if your linux is 32 bit  you should change 32-libleveldb.a to  libleveldb.a
                                     change 32-libmemenv.a to  libmemenv.a

run linux.sh 

cd ..

run autogen-polopoints-man.sh 

make
